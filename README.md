## Instructions

### Get Started

#### NPM

```shell script
git clone https://github.com/d-pollard/react-interview-preset
cd react-interview-preset
npm install
npm run start
```

From here, you can access the development site at http://localhost:3030

#### Yarn

```shell script
git clone https://github.com/d-pollard/react-interview-preset
cd react-interview-preset
yarn install
yarn start
```

From here, you can access the development site at http://localhost:3030
